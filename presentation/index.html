Deploy Collection (No-Code)

Route: localhost:3000/dashboard/deploy

How It Works

  • 1

    Fill the Form

    Enter collection name, symbol, max supply, mint price (in MON), max per wallet, and base URI for metadata.

  • 2

    Review Parameters

    A live preview card shows all your settings before deployment. Change anything before committing.

  • 3

    Click Deploy

    One click sends the transaction to Monad Testnet. MetaMask pops up for confirmation.

  • 4

    Get Your Contract

    After confirmation, you get the deployed contract address with a link to Monad Explorer.

No Solidity Required

The contract is pre-built with OpenZeppelin. You just configure the parameters and deploy. The ERC-721 contract includes public mint, whitelist mint, ownership, and withdrawal functions.

Allowlist Management

Route: localhost:3000/dashboard/allowlist

üë•

Add Addresses Manually

Paste wallet addresses one by one. Validates Ethereum address format (0x + 40 hex characters). Duplicates are automatically filtered.

üìÅ

CSV Upload

Upload a CSV file with wallet addresses. Handles comma and newline-separated formats. Bulk add hundreds of addresses instantly.

üå≥

Merkle Root Generation

Click "Generate Merkle Root" to compute the root hash from all addresses. Copy it and set it in the smart contract for on-chain whitelist verification.

Allowlist Flow

  • 1

    Collect Addresses

    Add manually or upload CSV with wallets from community, Discord, or Twitter giveaways.

  • 2

    Review & Clean

    View all addresses in a scrollable list. Remove any with one click. Export as CSV for backup.

  • 3

    Generate Merkle Root

    Compute the Merkle tree root hash. This is a single bytes32 value that's stored on-chain.

  • 4

    Set On-Chain

    Call setMerkleRoot() on your contract with the generated root to activate whitelisting.

Auto-Generated Code Snippets

Get plug-and-play code to restrict content based on NFT ownership. Four integration types provided:

üåê HTML / Vanilla JS

Example: React Gate

function TokenGate({ children }) { const { address } = useAccount(); const { data: balance } = useReadContract({ address: NFT_CONTRACT, abi: NFT_ABI, functionName: 'balanceOf', args: [address], }); if (balance > 0) { return children; // ‚úÖ Show gated content } return <p>You need an NFT to access.</p>; }

Step-by-Step

  • 1

    Install MetaMask

    Download from metamask.io. Available as a browser extension for Chrome, Firefox, Brave, and Edge.

  • 2

    Add Monad Testnet

    Open MetaMask ‚Üí Settings ‚Üí Networks ‚Üí Add Network. Enter the Monad Testnet details (see table).

  • 3

    Get Testnet MON

    Visit the Monad faucet to get free testnet MON tokens for deploying and minting.

    On NFT Terminal, click the purple "Connect Wallet" button. MetaMask prompts for approval. Your address shows in the navbar.
